# Skill Matrix

PanClaw v0.1 registers these initial domains:

| Domain | Skills |
|---|---|
| Messaging | personal WeChat official boundary, WeChat Official Account, WeCom, Feishu, Lark, DingTalk |
| Office/PDF | PyMuPDF, python-docx, openpyxl |
| Image | Stable Diffusion WebUI API, Pillow |
| Audio/Video | Whisper, FFmpeg |
| 3D | Blender Python |
| Desktop/Server | PyAutoGUI, Ansible |
| Hardware/Network | psutil, Prometheus |
| Cloud | Terraform, AWS, Alibaba Cloud, Tencent Cloud |
| Finance | AkShare, TuShare |
| Commerce/Life | Browser automation for price comparison, logistics query, food delivery draft |
| Health/Medical | BioPython, local medical RAG boundary |
| Learning | SymPy, Kiwix |
| Sports/Military | Scrapy, public RSS |
| AEC/GIS | FreeCAD, GDAL |
| Culture/Feng Shui | Lunar calendar adapter boundary |

The matrix is an execution boundary, not a claim that all upstream dependencies are installed. Every skill reports `dry_run`, `blocked`, `not_configured` or `ok` explicitly.
